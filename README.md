# EOS

EOS is the product and platform. The runtime intelligence that lives inside EOS is the **entity**. That entity can have its own chosen name, but that chosen name is **not** the product name.

If you are new here, start in the product folder:

- Product overview: [`EOS/README.md`](EOS/README.md)
- Recommended install path: [`EOS/docs/INSTALL.md`](EOS/docs/INSTALL.md)
- Recommended first-run flow: [`EOS/docs/QUICK_START.md`](EOS/docs/QUICK_START.md)

## Canonical default path

1. Run `setup\Setup-Full.ps1` for the recommended install.
2. Run `python verify.py`.
3. Start `launchers\start-standard.bat` for the recommended backend bundle.
4. Start `start-eos.bat` to bootstrap EOS and open the WebUI.

Deviate from that path only if you need a lighter install, manual model control, lower resource usage, or a custom backend mix.
