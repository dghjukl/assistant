Canonical Product Shape

EOS is the platform. The entity is the runtime intelligence inside EOS. The entity's chosen name is never the product name.

Operationally, this means:

  • You install, launch, package, and document EOS
  • You interact with and supervise the entity
  • You may configure or observe an entity name without renaming the product

Entry-Point Hierarchy

There are many valid scripts, but only one default hierarchy for new users.

  • setup\Setup-Full.ps1 — recommended install entry point
  • launchers\start-standard.bat — recommended backend launch entry point
  • start-eos.bat — recommended runtime bootstrap entry point
  • status-eos.bat — diagnostics without starting the WebUI

Deviate only for lower resource usage, manual model control, extra creativity, vision support, or exact backend/hardware selection.
