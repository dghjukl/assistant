Overview

What is EOS

EOS (Entity Operating System) is a local AI companion and assistant. Everything runs on your machine.

🧠
Local AI
Runs GGUF language models via llama.cpp. No cloud API required. No data leaves your machine.
🫞
Persistent Identity
Maintains identity and long-term memory across sessions. Does not reset between conversations.
🎙
Voice I/O
Local speech-to-text (Whisper) and text-to-speech (Piper/Amy). No cloud dependencies.
🔗
Integrations
Discord bot (optional), Google Calendar / Gmail / Drive as a first-class subsystem, and sandboxed computer use.
🌏
Worldview
Deposit source documents and EOS extracts a personalization profile for interpretive calibration.
🖥
Windows-Only
llama-server and Piper TTS are Windows executables. Linux/Mac not currently supported.

Reliability

Capability Degradation

When optional backends are missing, EOS degrades gracefully rather than failing hard.

The Three States

active
Service running and responding normally. Full capability available.
degraded
Service missing, intentionally dormant, or using a fallback path. Functionality continues with reduced immediacy, quality, or speed.
unavailable
Feature is cleanly disabled. EOS reports it as unavailable and does not attempt it.
