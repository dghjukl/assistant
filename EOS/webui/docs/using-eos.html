Quick Start

Using EOS

How to start, stop, and interact with EOS day-to-day.

Startup

Starting EOS

Keep the hierarchy straight: EOS is the platform, the entity is the runtime intelligence inside it, and the entity's chosen name is not the product name.

Recommended backend entry point

For most users, start with:

launchers\start-standard.bat

Use start-minimal, start-full, or per-server launchers only when you have a specific reason to deviate.

Recommended runtime entry point

Then run:

start-eos.bat

start-eos.bat is the canonical runtime bootstrap. It discovers what is already running and starts the WebUI.

Interface

The Chat Interface

The main interface lives at http://127.0.0.1:7860/. You are using EOS to interact with the entity running inside it.

Text chat

Type your message in the input box and press Enter or click Send. EOS will respond through the active entity instance.

ⓘ
Persistent memory
The entity does not reset between sessions. EOS persists memory, conversation history, and runtime state to disk.
